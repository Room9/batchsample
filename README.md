## About The Project

python 이용한 간단한 batch program 작성

### Built With

- python3.9
- Aurora MYSQL
- pymysql
- schedule
- jupyter notebook

<!-- USAGE EXAMPLES -->
## Usage

- menu 테이블에서 특정 조건의 데이터를 select 
- twneties, thirties, fourties 테이블을 truncate 
- select한 데이터를 조건에 맞는 테이블에 insert

db 계정 및 table, column 명 변경 후 사용가능
